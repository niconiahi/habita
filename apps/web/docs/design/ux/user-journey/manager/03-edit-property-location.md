# Manager: Edit property location

## The Lens

**Actor:** Property manager.

**Goal:** Update or correct a property's address.

**Scenario:** The manager needs to change the property's location — either because it was entered incorrectly during creation or because they want to update it.

---

## The Experience

### Phase 1: Update Location

**Routes:** `/admin/properties/[property_id]/edit`

**Actions:**

- Opens the property edit page
- Finds the location section
- Searches for the correct address
- Selects it from the suggestions
- Saves the change

**Thoughts:**

- "The address wasn't quite right"
- "Did the search find the correct location?"

**Emotions:** Corrective → Satisfied (location updated)

---

## The Output

### Pain Points

- Location update is on the same dense edit page as everything else — easy to miss

### Opportunities

- Highlight which sections have been filled vs. which are empty
