# Reservar visita a una propiedad

Para reservar una visita, primero debes subir tu informe crediticio. El mismo lo deberás subir desde [tu perfil](/profile).

Este informe es algo que de no hacer nosotros mismos, lo haría el administrador de la propiedad. Este documento en definitiva informa sobre tu situación crediticia.
