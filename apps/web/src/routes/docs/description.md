# Home

This is the landing page of the platform. Right now it simply points users toward the properties listing, which is the main entry point for browsing available rentals. Think of it as a lobby — you land here and get redirected to where the action is.

The global navigation bar is also set up here, showing notifications for property managers, the organization switcher for users who belong to multiple organizations, and a subscription warning banner if the user's plan is about to expire.
