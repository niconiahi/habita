# Browse properties

This is the main page where anyone can explore available rental properties. Users can narrow down what they're looking for using several filters: search by neighborhood or zone, filter by property features like number of rooms, bathrooms, bedrooms, total surface area, and construction year. They can also filter by tags (categories that describe the property) and by type of service offered.

Each property shows up as a card with a photo carousel, its location, and the monthly price. Clicking on a property takes you to its full detail page.
