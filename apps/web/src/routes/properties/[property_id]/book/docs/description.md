# Book a property visit

This is where tenants schedule a visit to see a property in person. The process has two steps:

First, you pick a date from the available options — these are days that the property manager has set up as open for visits. Once you select a date, the page shows the available time slots for that day.

Then you choose a specific time slot and confirm the booking. The slot gets reserved for you so no one else can take it. After booking, you're taken back to the property detail page.
