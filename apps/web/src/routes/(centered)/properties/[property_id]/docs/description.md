# Property detail

This is the public detail view for a specific property, shown to potential tenants browsing listings. It displays everything a prospective renter needs to know:

- A photo gallery with up to 5 images, expandable into a full viewer
- Property details organized in tabs: "Condiciones", "Propiedad" (dimensions, rooms), "Edificio" (building features), "Servicios", and "Otros" (amenities)
- Current pricing and location on a map
- Contact card for the property manager with WhatsApp link
- Organization card if the manager belongs to a real estate agency

Only published properties are visible here.
