# Panel del inquilino

Esta es la vista del inquilino de una propiedad que esta alquilando. Esta organizada en pestanas:

- **Contrato** — ver la fecha de vencimiento del contrato, archivos del contrato descargables, informacion de contacto del administrador con WhatsApp, precio actual del alquiler y periodos historicos de alquiler
- **Recibos** — subir y gestionar comprobantes de pago de alquiler, servicios e impuestos
- **Servicios** — ver los codigos de servicios publicos (electricidad, gas, agua, etc.)

Esta area solo es accesible para usuarios que son inquilinos de la propiedad.
