# Recibos del inquilino

Esta pestana permite a los inquilinos subir y ver sus comprobantes de pago — de alquiler, servicios publicos e impuestos. Los recibos se agrupan por mes, con el mes actual en la parte superior.

Solo el mes actual permite subir nuevos recibos. Los meses anteriores se muestran como historial de solo lectura. Los inquilinos pueden cargar meses anteriores desplazandose hacia abajo. Si necesitan eliminar un recibo, el sistema muestra un mensaje indicando que deben contactar a su administrador (usando el enlace de WhatsApp disponible en la pestana Contrato).
