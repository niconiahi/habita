# Pricing

This page displays the subscription pricing plans. Users can toggle between "Mensual" and "Anual" billing (annual saves 17%). Two tiers are offered:

- **Freelance** — $35/month (or $350/year) for independent property managers
- **Inmobiliaria** — $50/month per position (or $500/year) for real estate agencies

Each plan lists its included features and has an "Empezar gratis" button that links to the signup page.
