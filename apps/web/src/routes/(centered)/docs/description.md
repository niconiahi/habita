# Landing page

This is the marketing homepage of the platform. It introduces Habita to potential users with:

- A hero section with "Empezar gratis" to sign up and "Solicitar una demo" to request a demo via WhatsApp
- A problem section highlighting pain points with current real estate tools
- A solution section showing the 6-step workflow Habita offers
- A comparison table benchmarking Habita against 9 other real estate tools in the market
