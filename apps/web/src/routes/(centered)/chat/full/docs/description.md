# Chat (interno)

This is the internal version of the chat assistant, restricted to @habita.rent team members. It works the same way as the public chat but has access to the full technical documentation — route technical details, security architecture, broker system, error handling patterns, and digital signature API integration.

Team members can ask technical questions about the codebase, like how a specific action works, what tables a route queries, how authorization is enforced, or how the event broker processes messages. The assistant references specific file paths, function names, and database tables in its answers.

If you're not logged in, you'll be redirected to sign in. If your email isn't @habita.rent, you'll be sent to the public chat instead.
