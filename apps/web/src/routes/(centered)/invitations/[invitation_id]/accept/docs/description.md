# Aceptar invitación a un equipo

La página a la que llega un usuario cuando hace clic en el link de un email de invitación a un equipo. Si está logueado con la cuenta correcta, se une automáticamente al equipo y queda redirigido a la página del equipo. Si la invitación expiró, fue cancelada o el email no coincide con la cuenta logueada, ve un mensaje claro explicando qué pasó y qué puede hacer.

Si el usuario llega sin estar logueado, lo mandamos a iniciar sesión y después vuelve a esta misma página para completar el proceso.
