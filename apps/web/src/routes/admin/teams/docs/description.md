# Teams

This page is exclusively for real estate agency owners. It is the home for managing the agency's team structure:

- **Create team** — group managers under a named team (e.g. "Equipo Norte", "Sucursal Centro").
- **List teams** — see every team in the agency with its member count and creation date.
- **Drill into a team** — click a team to manage its members, send invitations, and see the properties each member is managing.

A default team named "Ejemplo" is created automatically with every new agency organization (and pre-seeded with sample managers + properties so the realtor can preview the experience before doing real work). It cannot be deleted but can be renamed, repurposed, or emptied of demo data.
