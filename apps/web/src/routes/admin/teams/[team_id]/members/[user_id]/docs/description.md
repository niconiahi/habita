# Team member detail

This page shows a single member of a team and the properties they are currently managing. From here the realtor can:

- **See member details** — name, email, and which team they belong to.
- **See managed properties** — every property where this user has manager access, with address and unit.
- **Reassign a property** — move a property from this member to another peer in the same team, or leave it unassigned. Reassigning is the prerequisite to removing this member from the team if they have properties in their care.
