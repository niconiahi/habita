# Team detail

This page shows a single team in the agency. From here the realtor can:

- **Invite by email** — send a pending invitation that, when accepted, will join the user to the agency as a manager and add them to this team.
- **See team members** — name, email, and how many properties each member is currently managing.
- **Remove a member** — remove a manager from this team. Blocked when the manager still has properties assigned (the realtor must reassign those properties first to keep the team-to-properties relationship intact).
- **Drill into a member** — click a member to see and reassign the properties they manage.

The default "Ejemplo" team comes pre-seeded with sample managers + properties so a new realtor can preview how the page looks before doing any real work.
