# Contract documents

This tab manages all files attached to the contract. Managers can see each file's type (contract PDF, signed PDF, etc.) and download them.

Files can be deleted by clicking "Eliminar" next to each file. The page also shows tenant insurance documents as a read-only list, pulled from the tenant's user profile.

If the contract PDF couldn't be generated due to missing property data, validation errors are displayed here to guide the manager on what needs to be fixed.
