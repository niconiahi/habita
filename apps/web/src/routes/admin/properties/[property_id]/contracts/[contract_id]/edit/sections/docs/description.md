# Contract sections

This tab is where the bulk of a contract's content is configured. Each section corresponds to a clause in the final contract document:

- **State** — manage contract items (furniture, appliances, etc.) with their condition and photos
- **Destiny** — select the property's intended use for this contract
- **Duration** — set contract start and end dates
- **Rental pricing** — configure escalation type and duration
- **Payment method** — enter the bank account (CBU) for payments
- **Late fees** — set the fine percentage for overdue payments
- **Returns** — set the percentage return amount
- **Early termination** — configure notice period in days
- **Showroom hours** — set how many hours the property can be shown during the contract
- **Warranty** — configure the guarantee type: property-backed, surety bond, or income-based with multiple guarantors
- **Jurisdiction** — select the court for dispute resolution
