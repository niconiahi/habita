# Contract sections

This tab is where the bulk of a contract's content is configured. Each section corresponds to a clause in the final contract document:

- **"Sección 2: estado"** — "Agregar item" to add contract items (furniture, appliances), "Guardar item" to save, "Eliminar item" to remove, "Agregar foto" to upload photos
- **"Sección 3: destino"** — select the property's intended use. "Guardar destino" to save
- **"Sección 6: plazo"** — set contract start and end dates. "Guardar plazo" to save
- **"Sección 7: canon locativo"** — configure escalation type and duration. "Guardar valores" to save
- **"Sección 8: forma de pago"** — enter the bank account (CBU). "Guardar CBU" to save
- **"Sección 9: mora"** — set the fine percentage. "Guardar porcentaje" to save
- **"Sección 14: devoluciones"** — set the percentage return amount. "Guardar porcentaje" to save
- **"Sección 15: recesión anticipada"** — configure notice period in days. "Guardar recesión anticipada" to save
- **"Sección 16: muestra de propiedad"** — set showroom hours. "Guardar cantidad" to save
- **"Sección 17: garantía"** — configure the guarantee type: property-backed, surety bond, or income-based. "Crear garantía" or "Guardar garantía" to save, "Agregar garante" for income-based guarantors
- **"Sección 21: jurisdicción"** — select the court. "Guardar jurisdicción" to save
