# Contract operations

This tab handles the final steps of a contract — generating the official PDF document and managing digital signatures.

- **PDF generation** — create a formatted PDF from the contract data
- **Digital signatures** — a two-phase workflow:
  1. Check if both landlord and tenant have digital certificates. If not, start the onboarding process for those who need one.
  2. Once both have certificates, send the contract for signing. Track each party's signature status and verify completion.

When both parties have signed, the contract is considered fully executed.
