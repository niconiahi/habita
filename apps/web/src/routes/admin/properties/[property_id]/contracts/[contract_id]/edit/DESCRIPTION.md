# Edit a contract

This is the most comprehensive contract management page. Here, managers can configure every aspect of a rental contract:

- **Contract details** — update the contract's core information like type, dates, and terms
- **Pricing periods** — set up and manage how the rent changes over time, create new pricing periods
- **Warranty** — set up the warranty/guarantee for the contract, including income guarantors who back the rental
- **Contract items** — add individual clauses or items to the contract, each with optional attached files
- **Documents** — upload and manage contract-related files (PDFs, scanned documents)
- **PDF generation** — generate a formatted PDF of the contract
- **Digital signatures** — check if landlord and tenant have digital certificates, start the onboarding process for digital signatures, send the contract for signing, and verify signatures

This page brings together everything needed to take a contract from draft to fully signed.
