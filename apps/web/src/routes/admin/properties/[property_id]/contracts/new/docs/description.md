# Crear un contrato para una propiedad

Aca es donde los administradores crean un nuevo contrato de alquiler para una propiedad especifica. Eligen el tipo de contrato y completan los datos necesarios. Una vez creado, se los redirige a la pagina de edicion del contrato donde pueden agregar todos los detalles como periodos de precio, informacion de garantia, y mas.

Se necesitan permisos de edicion sobre la propiedad para crear un contrato.
