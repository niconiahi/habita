# Edit a contract

This is the most comprehensive contract management area. It's organized into tabs that cover every aspect of a rental contract:

- **"secciones"** — edit the contract's clauses: state items with photos, destiny, pricing duration, payment method, fines, early termination terms, showroom hours, warranty details (property, surety, or income-based), and jurisdiction
- **"documentos"** — upload and manage contract-related files, view tenant insurance documents
- **"periodos"** — set up and manage how the rent price changes over time
- **"acciones"** — generate a formatted PDF of the contract, manage digital signatures (check certificates, start onboarding, send for signing, verify signatures)

This area brings together everything needed to take a contract from draft to fully signed.
