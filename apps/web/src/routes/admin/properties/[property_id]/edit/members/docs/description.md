# Property members

This tab shows everyone who has access to the property — managers, landlords, and tenants. Each member displays their avatar, name, role, and a green "Contactar" button that opens a WhatsApp conversation with that person (only shown if they have a phone number registered). This is the only place in the admin panel where a manager can contact a tenant or landlord via WhatsApp.

If no landlord is assigned yet, managers can click "Invitar dueño" to send an email invitation to a property owner so they can access the property's management.
