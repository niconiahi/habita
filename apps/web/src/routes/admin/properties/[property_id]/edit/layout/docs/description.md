# Property layout

This tab manages the physical layout of the property — its floors, rooms, and photos. Managers can:

- **Floors** — add floors up or down, reorder them by dragging, or remove them
- **Rooms** — add rooms to each floor, set their type and dimensions (width, length)
- **Photos** — upload and delete photos for each room
- **Room map** — switch to a visual map view where rooms can be dragged to position them spatially

A floor picker on the left lets managers navigate between floors. Each floor shows its rooms with their details and photos.
