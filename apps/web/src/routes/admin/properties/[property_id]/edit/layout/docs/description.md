# Property layout

This tab manages the physical layout of the property — its floors, rooms, and photos. Managers can:

- **Floors** — "agregar +" to add floors up or down, reorder them by dragging, or remove them
- **Rooms** — "agregar habitación +" to add rooms to each floor, "Guardar habitación" to save changes, "Eliminar habitación" to remove
- **Photos** — "Agregar foto" to upload photos for each room
- **Room map** — switch between "Dimensiones y fotos" and "Mapa" views. In map mode, rooms can be dragged to position them spatially, then saved with "Guardar mapa"

A floor picker on the left lets managers navigate between floors. Each floor shows its rooms with their details and photos.
