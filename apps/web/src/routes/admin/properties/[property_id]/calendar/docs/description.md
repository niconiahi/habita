# Calendario de visitas de la propiedad

Aca es donde los administradores configuran la disponibilidad para las visitas a la propiedad. Pueden crear turnos eligiendo una fecha y un rango horario, y los inquilinos veran esos turnos al reservar una visita.

Los turnos existentes tambien se listan aca, y los administradores pueden eliminar los que ya no sean necesarios. Solo las propiedades publicadas tienen calendario de visitas — si la propiedad aun no esta publicada, esta pagina no es accesible.
