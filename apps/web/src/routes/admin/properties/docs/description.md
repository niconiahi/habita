# Gestionar propiedades

Aca es donde los administradores y propietarios ven todas las propiedades a las que tienen acceso. Cada propiedad muestra su ubicacion, tipo (casa, departamento, etc.) y estado actual (en edicion, publicada, alquilada).

Desde aca pueden hacer clic en el boton "Nueva propiedad" para crear una, editar las existentes que aun estan en borrador, publicar propiedades para hacerlas visibles a los inquilinos, o despublicarlas para quitarlas del listado. Las propiedades publicadas tambien tienen un enlace para gestionar su calendario de visitas.
