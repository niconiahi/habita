# Crear una nueva propiedad

Aca es donde los administradores crean una nueva publicacion de propiedad. Necesitan completar tres datos: la ubicacion de la propiedad (usando una busqueda de direccion), el tipo de propiedad (departamento, casa, etc. — los departamentos tambien requieren un numero de unidad), y el destino de uso de la propiedad (residencial, comercial, etc.).

Una vez creada, la propiedad comienza en estado borrador y se puede seguir editando con ambientes, fotos, servicios y otros detalles antes de ser publicada.
