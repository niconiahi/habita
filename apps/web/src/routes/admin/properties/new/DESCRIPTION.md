# Create a new property

This is where managers create a new property listing. They need to fill in three things: the property's location (using an address search), the type of property (apartment, house, etc. — apartments also need a unit number), and what the property will be used for (residential, commercial, etc.).

Once created, the property starts in a draft state and can be further edited with rooms, photos, services, and other details before being published.
