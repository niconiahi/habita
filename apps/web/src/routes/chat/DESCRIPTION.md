# Chat with Habita

This is a conversational assistant that can answer questions about the Habita platform. Anyone can use it — no login required.

You can ask things like "what can tenants do?", "how does the contract signing work?", "what features does the admin dashboard have?", or "qué hace un inquilino para reservar una visita?". The assistant understands both Spanish and English.

It's powered by AI and has full knowledge of every feature, page, and user journey in the platform. Think of it as a smart guide to everything Habita can do.
