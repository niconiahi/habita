# ¿Qué datos se piden para un contrato?

## Datos obligatorios

- **Precio inicial**: el canon locativo mensual con el que se publica la propiedad
- **Plazo**: fecha de inicio y finalización del contrato
- **Tipo de escalación**: cómo y cada cuánto se ajusta el precio (ICL, IPC, etc.)

## Datos opcionales

- **Destino del inmueble**: uso específico dentro de los permitidos por la propiedad
- **CBU**: cuenta bancaria para recibir pagos
- **Mora**: porcentaje de penalización por atraso en pagos
- **Devoluciones**: porcentaje de devolución
- **Recesión anticipada**: días de preaviso requeridos
- **Muestra de propiedad**: horas permitidas para mostrar el inmueble
- **Garantía**: propiedad, caución o recibo de sueldo con datos del garante
- **Jurisdicción**: tribunal competente

Todos estos datos se utilizan para generar el contrato en formato PDF, listo para firmar digitalmente.
