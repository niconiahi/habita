# ¿Qué datos pedimos a cada usuario?

## Datos obligatorios

- **Nombre y apellido**
- **Email**: utilizado para acceso y notificaciones
- **Tipo de cuenta**: dueño/inmobiliaria o inquilino

## Datos opcionales pero recomendados

- **Teléfono**: para comunicación directa
- **DNI**: necesario para la firma digital del contrato
- **Foto de perfil**

## Datos adicionales para inquilinos

- **Informe crediticio**: requerido para agendar visitas a propiedades

## Seguridad de tus datos

Sabemos que pedimos información sensible. Por eso, toda la información personal que guardes en tu perfil se [guarda de forma segura](/learn/user-information).
