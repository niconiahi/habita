# ¿Qué datos incluye una propiedad?

## Datos obligatorios

- **Ubicación**: dirección completa con geolocalización
- **Destino**: residencial, comercial, u otros usos permitidos

## Datos opcionales

- **Fotos**: imágenes de la propiedad para la publicación
- **Servicios**: gas, electricidad, agua, internet y sus identificadores
- **Tags**: características como balcón, cochera, ascensor, pileta, etc.
- **Edificio**: año de construcción
- **Disposición**: pisos, ambientes con dimensiones y mapa de distribución

Cuantos más datos cargues, más completa será la publicación y mejor podrán evaluar los candidatos si la propiedad es lo que buscan.
