# Asistente de Habita

Este es el endpoint que alimenta al asistente de la plataforma. Cuando un usuario hace una pregunta, el asistente busca en la documentacion de la plataforma usando herramientas para encontrar informacion relevante y luego formula una respuesta basada en lo que encontro.

El asistente puede responder preguntas sobre funcionalidades de la plataforma, guiar a los usuarios a paginas especificas y explicar flujos de trabajo — todo en el idioma del usuario (espanol o ingles). Nunca inventa funcionalidades ni pasos de navegacion.
