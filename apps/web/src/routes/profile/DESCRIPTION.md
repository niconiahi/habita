# User profile

This is where users manage their personal information and documents. They can update their name, surname, phone number, national ID number (documento), and CUIL. The email field is shown but can't be changed.

Users can also upload important documents here — things like their ID card, proof of income, or any other file that might be needed for signing rental contracts. All uploaded documents are listed in a table where they can be downloaded.

There's a helpful link to a "learn more" article that explains what information is needed and why.
