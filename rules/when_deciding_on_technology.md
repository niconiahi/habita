# When deciding on technology

Whenever having the need to consume _any_ technology. I will always put above all that:
- Only open source options. I don't want to pay a penny
- Dockerized solutions
- Always prefer writting from scratch if it's straightforward. The only things that shouldn't be written from scratch are really heavy and complicated stuff, like in the case of this project: Nominatim, a KV store. And in the future: image transformation 
- Put on a pedestal being relunctant to add a depedency of _any_ sort: payed or nor payed
