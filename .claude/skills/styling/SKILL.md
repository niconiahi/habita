This skill guides how styling should be implemented across the application

1. Preference of writting CSS as native `.css` files. We could apply some extra `className` Tailwind's class only for layout classes like `mt-4` or `pl-6` or similar
2. Always create such CSS file next to the file being applied. If the file is located at `/components/wheel.tsx`, then the CSS file should be created at `/components/wheel.css`
